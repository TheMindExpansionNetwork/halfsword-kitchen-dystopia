# Ratatouille: Dystopia Edition - Easy Install

One-folder install. Copy → Paste → Play. The mod is on when you launch the game.

## Super Simple Install (3 steps)

1. Download this repo as ZIP and extract it.

2. Copy the entire `HalfswordUE5` folder.

3. Paste it into your game root:
   ```
   I:\SteamLibrary\steamapps\common\Half Sword\
   ```
   (Merge folders when asked)

4. (Important) Also copy these two files into your game:

   - `mods.txt` → replace the one in:
     `...\HalfswordUE5\Binaries\Win64\UE4SS\Mods\mods.txt`

   - `UE4SS-settings.ini` → replace the one in:
     `...\HalfswordUE5\Binaries\Win64\UE4SS\UE4SS-settings.ini`

5. Restart Half-Sword.

The mod is now enabled by default.

Open the console with **Insert** and type:
- `KitchenDystopia` (full game changing mode)
- `PanTime`, `IronChef`, etc.

No more editing files after this.

---

**Note**: This replaces your `mods.txt` and `UE4SS-settings.ini`. Back them up first if you have other mods.

The visual kitchen props will appear in the world automatically.
